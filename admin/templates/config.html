Upload your Google Cloud service account credentials file to enable Vision deposit screenshot verification.
How to set up Google Vision:
  1. Go to Google Cloud Console and create or select a project.
  2. In the left menu go to APIs & Services → Library, search for Cloud Vision API and click Enable.
  3. Go to APIs & Services → Credentials and click Create Credentials → Service Account.
  4. Give it a name (e.g. chatbot-vision), click Create and Continue, skip optional steps, then click Done.
  5. Click on the new service account, go to the Keys tab, click Add Key → Create new key, choose JSON and click Create.
  6. A credentials.json file will download automatically — upload it using the field above.
  7. The Vision API has a generous free tier (1,000 requests/month). No billing is required for typical usage.

Need more help? See the official Google Cloud Vision setup guide.
